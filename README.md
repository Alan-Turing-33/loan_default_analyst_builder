These files implement the loan default project as published in the Analyst Builder community. 

This paper analyses a dataset issued by AnalysisHorizon Financial Group with over 600 personal loans across 2024 and 2025. 
The company has noticed that roughly 1 in 4 loans are defaulting, which is well above their target of 12%. 
​

The overall average  default rate was 24.3 %.
​

If we only consider loans with a credit score larger than 625 and an interest_rate smaller than 13.5 and a 
DTI ratio smaller than 60, then we can reduce the default percentage to 12.97% 
​

This project was executed in google colab using python with the modules polars and seaborn with additional google sheets
